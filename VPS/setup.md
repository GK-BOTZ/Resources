## CHECK AND UPDATE SYSTEM PACKAGES

```bash
sudo apt update -y && sudo apt upgrade -y
```


## INSTALL REQUIRED PACKAGES 

```bash
sudo apt install -y \
python3 \
python3-pip \
git \
nano \
zip \
unzip \
screen \
ffmpeg \
curl 
```


### Note: Aap Khudse Aur Packages Add Kar Sakte Hai.

Like: docker install ke liye.

```bash
sudo apt install docker.io
```
